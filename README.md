# What is DamEdit-CLI?

- It's a simple terminal-editor witten in python but usefull.
- It uses `curses` to show/edit/change texts in terminal.
- It has a `config.json` file to change its colors

---
# Installation

To run editor :
```bash
python main.py <your_file>
```
